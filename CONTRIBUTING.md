# Contributing to pi-workspace-gate

Thanks for your interest in contributing! This is a small, focused extension so contributions are welcome in the form of:

- Bug reports
- Feature suggestions
- Code improvements (typos, pattern coverage, edge cases)

## Getting Started

1. Fork and clone this repo
2. Test locally with `pi -e ./extensions/pi-workspace-gate`
3. Make your changes and submit a pull request

## Guidelines

- Keep the extension lightweight — it's a security gate, not a feature bloat
- Follow existing code style (no semicolons, no explicit type annotations unless necessary)
- New patterns should be well-reasoned and cover real-world use cases
